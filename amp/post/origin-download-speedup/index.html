国内Origin下载加速

Sun, Nov 18, 2018   dyzdyz010
# tutorial
# game
# general

打开 Origin 安装目录下（默认C:\Program Files(x86)\Origin）的 EACore.ini 文件，复制粘贴如下内容：

[connection]
EnvironmentName=production

[Feature]
CdnOverride=akamai

保存并关闭，重启Origin，本人测试下载可以达到满速。
